== Asteroids Game ==

Prerequisites:
* Gosu - sudo gem install gosu

To Play:
* ruby asteroids_game_window.rb

Controls:
* Left Arrow    - Turn Left
* Right Arrow   - Turn Right
* Up Arrow      - Move Forward
* Space         - Fire Bullet
* Left Shift    - Hyperspace (randomly relocates you on screen)
